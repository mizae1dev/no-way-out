# Setup Guide

This is the setup guide for the project. Follow these steps to set up the environment and start working with the application.

## Prerequisites

- Ensure you have [Node.js](https://nodejs.org/) installed.
- Other dependencies required for the project.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mizae1dev/no-way-out.git
   ```

2. Navigate to the project directory:
   ```bash
   cd no-way-out
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```
   
## Running the Application

To run the application, use the following command:
```bash
npm start
```
